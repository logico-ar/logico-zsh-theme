# Logico Zsh theme.
A minimal Oh-my-zsh theme.

## Features

- Fast.
- Integration with git.
- Better indicator for `vi-mode`.
- Indicator for remote sessions.

## Install.

- Download or clone this repo on ~/.oh-my-zsh/themes
